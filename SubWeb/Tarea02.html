AUTOEVALUACION 02

Pregunta 01: Una configuración opcional enviada por el servidor DHCP es información de:

  • DNS
  • IP
  • FTP
  • MAC
  • SSL

Respuesta: DNS (Domain Name System). Además de asignar direcciones IP, un servidor DHCP puede enviar información adicional como la dirección de los servidores DNS, la puerta de enlace predeterminada (gateway) y las máscaras de subred.

Pregunta 02: A la suplantación de DHCP se le conoce como:

  • Spoofing
  • Snooping
  • Adware
  • Spyware
  • Malware

Respuesta: Spoofing. El DHCP spoofing es un tipo de ataque en el que un atacante configura un servidor DHCP no autorizado en la red para distribuir direcciones IP falsas u otra información incorrecta a los clientes DHCP, con el fin de interceptar o manipular el tráfico de red.

Pregunta 03: Las IPS de equipos de red se configuran en DHCP como direcciones:

  • Excluidas
  • Externas
  • Válidas
  • Grupales
  • Cedidas

Respuesta: Excluidas. En DHCP, las direcciones excluidas son aquellas que se configuran para no ser asignadas automáticamente por el servidor DHCP. Estas direcciones suelen ser utilizadas para dispositivos de red como routers, switches, servidores y otros equipos que requieren una dirección IP estática.

Pregunta 04: Sin DHCP las computadoras de una red deben de configurar sus direcciones IP de forma:

  • Estática
  • Aleatoria
  • Dinámica
  • Semanal
  • Ocasional

Respuesta: Estática. En ausencia de un servidor DHCP, cada dispositivo en la red debe tener su dirección IP configurada manualmente (estática) para asegurarse de que no haya conflictos y que cada dispositivo pueda comunicarse correctamente en la red.

Pregunta 05: Si un router asigna direcciones IP que no se deberían asignar, esto se evita mediante:

  • Snooping
  • Antivirus
  • Firewall
  • Directivas
  • Spoofing

Respuesta: Snooping. El DHCP snooping es una técnica utilizada para proteger la red contra servidores DHCP no autorizados (maliciosos) que intentan asignar direcciones IP incorrectas. Esta técnica permite que los switches supervisen las comunicaciones DHCP y bloqueen cualquier respuesta de servidores DHCP no autorizados.
