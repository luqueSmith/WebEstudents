Autoevaluación T04

Pregunta 01: Para activar el servicio FTP en un servidor Windows Server, el rol correspondiente se encuentra en:

  • Roles de IIS
  • Roles de FTP
  • Servidor FTP
  • Servidor Web
  • FTP Windows

Respuesta: Roles de IIS (Internet Information Services). Dentro del rol de IIS, puedes agregar el servicio FTP para habilitar y configurar servidores FTP en el servidor Windows.

Pregunta 02: Si el administrador de red configura manualmente las rutas en el router, a estas rutas se les llama:

  • Estáticas
  • Directas
  • Rápidas
  • Dinámicas
  • Completas

Respuesta: Estáticas. Las rutas estáticas son configuradas manualmente por el administrador y no cambian automáticamente en respuesta a cambios en la red, a diferencia de las rutas dinámicas que se ajustan automáticamente mediante protocolos de enrutamiento.

Pregunta 03: BGP es un ejemplo de un enrutamiento:

  • Dinámico
  • General
  • Básico
  • Completo
  • Estático

Respuesta: Dinámico. BGP es un protocolo de enrutamiento dinámico utilizado para intercambiar información de enrutamiento entre sistemas autónomos en Internet. Permite que los routers actualicen automáticamente las rutas basadas en la información de enrutamiento recibida de otros routers BGP.

Pregunta 04: ¿Qué significa FTP?

  • File Transfer Protocol
  • Forward Time Protocol
  • File Terminal Protocol
  • File Transfer Personal
  • Forward Term Protocol

Respuesta: File Transfer Protocol. Es un protocolo estándar de la red utilizado para la transferencia de archivos entre un cliente y un servidor en una red.

Pregunta 05: Se agrega un certificado SSL a un servicio FTP por:

  • Seguridad
  • Registro
  • Importancia
  • Costumbre
  • Integridad

Respuesta: Seguridad. El certificado SSL (Secure Sockets Layer) se utiliza para cifrar la conexión entre el cliente y el servidor FTP, asegurando que los datos transferidos sean confidenciales y seguros.
