Autoevaluación T03

Pregunta 01: Para poder crear proyectos de páginas web con IIS se utiliza:

  • IIS Manager
  • Net framework
  • Admin service
  • Server admin
  • Default web

Respuesta: IIS Manager. IIS Manager es la herramienta de administración que permite configurar y gestionar sitios web, aplicaciones y servicios en un servidor IIS.

Pregunta 02: Un ataque de secuestro DNS está dirigido hacia:

  • Caché
  • Cliente
  • Router
  • Servidor
  • Switch

Respuesta: Caché. En un ataque de secuestro de caché DNS (DNS cache poisoning), el objetivo es manipular la caché del servidor DNS para que resuelva nombres de dominio en direcciones IP incorrectas, redirigiendo el tráfico a sitios maliciosos o no deseados.

Pregunta 03: El servicio web por defecto de Windows Server es:

  • IIS
  • TLS
  • ASA
  • SSH
  • SSL

Respuesta: IIS (Internet Information Services). IIS es el servidor web y de aplicaciones de Microsoft que se incluye con Windows Server para alojar sitios web y servicios web.

Pregunta 04: Al servidor que contiene los DNS récords asociados se le llama:

  • TLD
  • RCP
  • RTP
  • DHCP
  • SSL

Respuesta: TLD (Top-Level Domain). El servidor TLD contiene registros relacionados con los dominios de nivel superior (como .com, .org, .net) y ayuda a resolver los nombres de dominio hacia los servidores de nombres autoritativos para dominios específicos.

Pregunta 05: Un envenenamiento en servidores DNSs introduce datos falsificados en:

  • Caché
  • Registro
  • Tablas
  • Rutas
  • Router

Respuesta: Caché. En un ataque de envenenamiento de caché DNS (DNS cache poisoning), el objetivo es insertar información incorrecta en la caché de un servidor DNS para redirigir el tráfico a sitios maliciosos o incorrectos.
