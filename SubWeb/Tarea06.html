Autoevaluación T06

Pregunta 01: Cada usuario de Exchange puede tener como máximo de buzones:

  • 1
  • 4
  • 5
  • 2
  • 3

Respuesta: 1 buzón. En Microsoft Exchange, cada usuario tiene un buzón principal asociado con su cuenta. Sin embargo, los usuarios pueden tener buzones adicionales, como buzones compartidos o buzones de recurso, que no cuentan como buzones personales.

Pregunta 02: Para crear un nuevo buzón de correo se utiliza:

  • Exchange Admin Center
  • Active Directory for EAC
  • Exchange Active Directory
  • Linked Mailbox Exchange
  • Mailbox Exchange Center

Respuesta: Exchange Admin Center (EAC). El Exchange Admin Center es la interfaz gráfica de administración basada en web que permite a los administradores gestionar buzones de correo, configuraciones de Exchange y otros aspectos relacionados con el correo electrónico.

Pregunta 03: La ubicación predeterminada de la cuenta al crear un usuario es el contenedor denominado:

  • Usuarios
  • Exchange
  • Mailbox
  • Invitados
  • Outlook

Respuesta: Usuarios. En Active Directory, por defecto, los nuevos usuarios se crean en el contenedor "Usuarios", a menos que se especifique una ubicación diferente durante el proceso de creación.

Pregunta 04: Una de las características que cuenta Exchange Server es:

  • Antispam
  • Kaizala
  • Office365
  • Messenger
  • Telefonía

Respuesta: Antispam. Exchange Server incluye funciones de filtrado antispam para proteger los buzones de correo de los mensajes no deseados y potencialmente dañinos.

Pregunta 05: Al agente de transferencia de correo se le conoce como:

  • MTA
  • SLA
  • IMAP
  • SSL
  • POP

Respuesta: MTA (Mail Transfer Agent). Un MTA es responsable de recibir, enrutar y entregar correos electrónicos entre servidores de correo.
