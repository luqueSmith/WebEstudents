AUTOEVALUACION 01

Pregunta 01: Un servidor Web es un ejemplo de:

  • Cliente Servidor
  • Gestión principal
  • Maestro esclavo
  • Base de datos
  • Controlador DC

Respuesta: Cliente servidor. El modelo cliente-servidor es una arquitectura de red en la que un servidor proporciona recursos o servicios a uno o más clientes.

Pregunta 02: Protocolo que permite la comunicación entre el Servidor de Archivos y los clientes:

  • DNS
  • SSH
  • SSL
  • SMB
  • STP

Respuesta: SMB. SMB es un protocolo de red que permite compartir archivos, impresoras y otros recursos entre nodos de una red, principalmente en redes Windows.

Pregunta 03: ¿Cómo se llama al intermediario entre la computadora y la impresora?

  • Servidor de impresión
  • Impresora compartida
  • Cola para impresión
  • Switch de networking
  • Cableado estructurado

Respuesta: Servidor de impresión. Un servidor de impresión gestiona las tareas de impresión de varias computadoras y las envía a una o más impresoras conectadas a la red.

Pregunta 04: Gestiona la autenticación y acceso a red para los usuarios de la organización:

  • Domain controller
  • Name services
  • Security server
  • LDAP controller
  • Policy services

Respuesta: Domain controller (controlador de dominio). Un controlador de dominio es un servidor en una red que responde a las solicitudes de autenticación y verifica los usuarios y contraseñas, gestionando el acceso a los recursos de la red en un dominio de Windows.

Pregunta 05: Brinda archivos a disposición de un cliente conectado a la red:

  • File server
  • SSH server
  • DC Windows
  • DC service
  • Data base

Respuesta: File server (servidor de archivos). Un servidor de archivos almacena y gestiona los archivos de datos para que los usuarios de una red puedan acceder a ellos, compartirlos y gestionarlos de manera centralizada.
