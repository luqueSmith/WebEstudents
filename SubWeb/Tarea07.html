Autoevaluación T07

Pregunta 01: Puede utilizarse para saltar las restricciones de navegación de una empresa:

  • Proxy
  • Módem
  • Router
  • Enlace
  • Worm

Respuesta: Proxy. Un servidor proxy actúa como intermediario entre los usuarios y la red, permitiendo a los usuarios eludir restricciones de navegación, acceder a contenido bloqueado y enmascarar su identidad en línea.

Pregunta 02: Una característica de usar un servidor proxy es:

  • Compartir Internet
  • Uso de antispam
  • Configurar VLANs
  • Tablas de ruteos
  • Usar enlaces VPN

Respuesta: Compartir Internet. Un servidor proxy puede permitir que varios usuarios compartan una única conexión a Internet, además de proporcionar funciones adicionales como filtrado de contenido y control de acceso.

Pregunta 03: Un protocolo de tunelización de red privada virtual VPN usada en Windows Server es:

  • IKEv2
  • TLSv1
  • DES2
  • SSLv1
  • SSH2

Respuesta: IKEv2 (Internet Key Exchange version 2). IKEv2 es un protocolo de tunelización que proporciona seguridad y estabilidad en las conexiones VPN, y es ampliamente utilizado en Windows Server para la configuración de VPN.

Pregunta 04: Para usar un servicio proxy en una computadora se requiere configurar:

  • IP y puerto del proxy
  • MAC e IP del proxy
  • Nombre de dominio
  • IP Gateway y DNS
  • IP de DNSs y WINS

Respuesta: IP y puerto del proxy. Para que una computadora pueda conectarse a través de un proxy, es necesario especificar la dirección IP del servidor proxy y el puerto que utiliza para la comunicación.

Pregunta 05: VPN significa:

  • Virtual Private Network
  • Virtual Prime Networking
  • Virtual Personal Network
  • Vinculated Private Net
  • Vault Private Networking

Respuesta: Virtual Private Network. Una VPN crea una conexión segura y cifrada a través de una red pública, como Internet, permitiendo que los datos viajen de forma segura entre un dispositivo y un servidor o entre varias redes privadas.
